# Network-Standards
<p>Реализация программ для взаимодействия между процессами, а также в локальной сети при помощи таких стандартов как Pipes, MailSlots, Sockets, MSMQ в виде чата.</p>
<p>Реализация веб-сервиса, который может осуществлять следующие операции:
<ol>
  <li>Шифрование и дешифрование строки;
  <li>Сложение двух массивов поэлементно;
  <li>Удаление дубликатов, сортировка, поиск минимума и максимума в массиве;
  <li>Поиск самого частого слова в строке, а также ее перевод в верхний регистр.
</ol>
